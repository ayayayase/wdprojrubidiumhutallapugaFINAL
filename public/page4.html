About Game

about the game and creator !!

The website is be a dress-up game in which users can create their own characters by clicking on buttons.
It includes a homepage in which users can input their name, the main dress up page which allows the user to go through a selection of features and use any of their choice for the character, and finally a "picture" page that will display the finished character and a dialogue box with the user's name and a background of their choice, so they could screenshot it and use it for any purpose they wish (profile picture, background picture, profile banner, etc).
